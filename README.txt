# TestAdept
 
Данная тестовая программа была созданна для сортировки, добавления и удаления объектов.

Программа была реализована примерно за 7 часов.

 После запуска программы, вы увидите список объектов и их параметров. Ниже будут доступна панель опций для манипуляций с приложенной БД.

 "Новый объект" 
 - Появляется окно диалога с четырьмя параметрами будущего объекта. Три из них заполняются вручную, а четвертый можно выбрать из предложенных вариантов. Так же есть кнопка подтверждения и отмены.

 "Полный список" 
 - Обновление БД, таблица возвращается к списку объектов из БД, отменяя все изменения что произошли без сохранений.

 "Удалить" 
 - Выделенный объект в таблице после нажатия будет удален.

 "Сортировка" 
 - Появляется окно диалога, где предоставлен выбор из четырех сортировок. По дальности, имени, типу и времени создания. Дополнительные элементы некоторых из способов сортировки разблокируются после выбора. При нажатии на кнопку подтверждения, окно диалога закроется, а в таблице остануться объекты кторорые прошли условие сортировки. Вернуть полный список объектов можно на кнопку "Полный список".

 "Сохранить" 
 - Данные таблицы будут перезаписаны в файл БД.
